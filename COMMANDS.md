# Commands

Install dependencies:

```bash
npm install
```

Create a local environment file:

```bash
cp .env.example .env
```

Run the default script version:

```bash
npm start
```

Run version 1:

```bash
npm run start:v1
```

Run version 2:

```bash
npm run start:v2
```

Run version 3:

```bash
npm run start:v3
```

Remove local runtime history:

```bash
rm -f checked-users.txt
```
